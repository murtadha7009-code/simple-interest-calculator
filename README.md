# Simple Interest Calculator

This repository is part of the final project for the GitHub UI and Git CLI assignment.
It includes a simple Bash script (simple-interest.sh) for calculating simple interest.

## Files Included
- LICENSE (Apache 2.0)
- README.md
- CODE_OF_CONDUCT.md
- CONTRIBUTING.md
- simple-interest.sh

## Usage
The script calculates simple interest using the formula:
P * R * T / 100
